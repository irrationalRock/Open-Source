Welcome! How to use this app?

To use this app, there are two API endpoints you can follow

By Param GET Request

https://osd600.herokuapp.com/api/phonenumbers/parse/text/[phone number you want to validate and format]
Example: .../api/phonenumbers/parse/text/Seneca Phone 416-491-5050
And it should return an array, containing a formatted phone number.
If the phone you submitted in not valid, an error message will show accordingly

Upload a Text file

Upload a text file HERE

NOTE: You can use the text file provided on my GitHub named "numbers.txt"

It will return an array of formatted valid Phone Numbers
